## Start
npm run develop

## Install a new font
- npm i @fontsource/playfair-display
- Import it in js file like `import "@fontsource/playfair-display/700.css"`

## Deploy
- npm run deploy

## Add a new page
- Create a new .md file in src/markdown-pages
- Add the following header:

```yml
---
slug: "/landing"
date: "2019-05-04"
title: "Landing"
summary: "Small summary"
---
```
